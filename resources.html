Frequently Asked Questions

Common Issues

  • Why won't my Arduino connect?

    Make sure you're using Chrome or Edge browser (Safari and Firefox don't support Web Serial API). Also verify your Arduino is plugged in via USB and you've selected the correct port in Arduino IDE before uploading the code.

  • The servos aren't moving correctly. What should I do?

    Check that your servos are connected to the correct pins: LEFT servo on pin 10, RIGHT servo on pin 9. Also verify your power supply is adequate â€” servos need sufficient current to operate properly.

  • Which Arduino boards are compatible?

    Braillino works with Arduino Uno and Arduino Nano. Make sure to select the correct board type in Tools â†’ Board before uploading the sketch.

  • Can I use this on mobile devices?

    Unfortunately, no. The Web Serial API is only available on desktop versions of Chrome and Edge. Mobile browsers don't currently support serial port communication.

  • What characters are supported?

    Braillino supports Aâ€“Z (uppercase and lowercase), and punctuation marks: . , ? ! ; : - ' . Spaces are also supported.

  • How do I calibrate the servo positions?

    The servo positions are pre-calibrated in the Arduino code. If you need to adjust them for your specific hardware, modify the LEFT_POS and RIGHT_POS arrays in the .ino file.

  • The servos show letters too fast/slow. Can I change the speed?

    Yes! In the Arduino code, find DISPLAY_TIME = 4000 and change the value (in milliseconds). For example, use 6000 for 6 seconds per character, or 2000 for 2 seconds.

  • Do I need to reconnect every time I navigate between pages?

    No! Once connected on the main page, your connection status is remembered as you browse. When you return to the main page, Braillino will automatically reconnect to your previously used port.

  • How do I get help or report a bug?

    Check the About page for contact information, or visit the project repository if available. Make sure to describe the issue clearly and include your browser version and Arduino board type.
